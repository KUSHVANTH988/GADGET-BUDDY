# Project Name

This repository contains a static page `open.html`.

## Quick ways to open `index.html`

### 1) Open directly on GitHub
Click the file in this repository:  
`open.html` → View → **Raw** or **Open**.  
(Or click the link in this README.)

### 2) Serve it with GitHub Pages (recommended — auto-serve in browser)
To make the site load automatically in the browser, enable **GitHub Pages**:

1. Go to your repository Settings → Pages.
2. Under **Source**, choose branch `main` (or `gh-pages`) and folder `/ (root)`.
3. Save. GitHub will give you a Pages URL like `https://<username>.github.io/<repo>/`.
4. To have that URL open `open.html` automatically, either:
   - Rename `open.html` to `index.html` (recommended), **or**
   - Add an `index.html` that redirects to `open.html` (example below).
